# AWS Essentials

Technical documentation covering AWS fundamentals and core services.

## Overview

This repository is where I keep what's worth documenting — organized as a technical reference I can revisit.

## Scope

Topics covered include:

- Cloud computing fundamentals
- AWS global infrastructure
- AWS accounts and IAM
- EC2 instance types and pricing models
- AMIs and Security Groups
- EBS, snapshots, and S3 storage concepts
- Lifecycle management
- Infrastructure as Code with AWS CloudFormation
- Workflow orchestration with AWS Step Functions
